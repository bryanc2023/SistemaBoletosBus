Acerca de BookMyDrive

Bienvenido a tu aplicación de reserva de boletos de buses. Como pasajero, tienes acceso a funciones diseñadas para gestionar eficientemente todos tus reservas. Desde este panel, puedes visualizar viajes proximos y buscarlos según preferencias.

Al cancelar no se aplica valores recaudos o extras si se lo realiza dentro del periodo de tiempo establecido, pero se le penaliza dentro de la aplicación
